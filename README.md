## How to Install

1. **Install a Userscript Manager:**
   First, download and install the Tampermonkey extension for your browser:
   * [Tampermonkey for Chrome / Edge / Brave](https://www.tampermonkey.net/)
   * [Tampermonkey for Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

2. **Install the Script:**
   Once Tampermonkey is installed, click the link below to install the script:
   * 👉 [**Click Here to Install browse-maxing**](https://raw.githubusercontent.com/MiloSejuk/browse-maxing/main/main.user.js)

3. **Confirm Installation:**
   A Tampermonkey tab will open. Click the **Install** button. Enjoy!
